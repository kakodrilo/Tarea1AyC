TAREA 1 ALGORTIMOS Y COMPLEJIDAD

* Ejecución:
- Abrir terminar de linux en la carpeta donde esté el makefile y el archivo MIB.cpp
- En la misma carpeta debe ir el archivo de las entradas input.txt, ya que al ejecutar se leerán las entradas desde
  ese archivo. 

- Para compilar y ejecutar el programa se debe ejecutar el siguiente comando en terminal:
                                    $ make
- Para borrar el ejecutable y archivo .o, se debe ejcutar en la terminal:
                                    $ make clear

* Consideración:
- El programa se probó utilizando el compilador g++ versión 7.4.0 en una terminal de ubuntu versión 18.04.1
- Dado el contexto actual, no se pudo probar el programa en los computadores del LDS.
